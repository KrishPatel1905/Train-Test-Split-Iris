# 🌸 Iris Classification using Machine Learning

## 📌 Project Overview

This is a beginner-level machine learning project where I trained a model on the Iris dataset using a Random Forest Classifier.

The goal of this project is to understand the basic workflow of machine learning:

* Loading data
* Training a model
* Making predictions

---

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn

---

## 📊 Dataset

This project uses the Iris dataset, which contains information about different types of iris flowers.

Features include:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target:

* Flower species

---

## 🤖 Model Used

* Random Forest Classifier

---

## ⚙️ How it Works

1. Load dataset using pandas
2. Separate features (X) and target (y)
3. Train model using Random Forest
4. Predict output using trained model
